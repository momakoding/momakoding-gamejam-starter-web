$ cd YOUR_REPO
# install dependencies
$ pnpm install

$ pnpm dev

# build for production → dist/
$ pnpm build
UI Framework│Vue 3 SFC · <script setup lang="ts">
Game Engine│Phaser 4 · Arcade Physics
Language│TypeScript strict · any forbidden
Bundler│Vite 8 · @ alias → ./src
Styling│Tailwind CSS v4 + tw-animate-css
Routing│vue-router 4 · Hash History
State│Pinia + persisted-state
Icons│lucide-vue-next
Utilities│@vueuse/core · tailwind-merge · animate.css
src/
├── engine/ ① UI-agnostic Phaser wrapper
│ ├── game-shell/ GameShell lifecycle
│ └── event-bus/ Map+Set pub/sub
├── contents/ ② Single source of truth
│ ├── constants.ts ★ SCENE_KEYS / EVENT_KEYS / GAME_CONFIG
│ ├── game-info/ ★ title · gameplay copy · team info
│ └── scenes/ BootScene / GameScene / ...
├── runtime/ ③ Vue-side module singletons
│ ├── game.ts useGame()
│ └── event-bus.ts useEventBus()
├── composables/ ④ True Vue composables (Ref / lifecycle)
├── components/ game-button · mmkd-starter-credit
└── pages/ home · how-to-play · about · game
