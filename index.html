快速开始

Quick Start

terminal 
# 1. Use this template on GitHub, then clone your repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO

# 2. Install dependencies
pnpm install

# 3. Start dev server
pnpm dev      # http://localhost:5173/

# 4. Build for production
pnpm build    # output → dist/
